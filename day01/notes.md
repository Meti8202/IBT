# Terminal Commands
1. ls = lists files and folders in the directory 
2. cd = change directory  
3. pwd = shows the path of the directory user is in 
4. mkdir = creates a new directory 
5. touch filename = create new empty file
# Git Commands
1. git init = initializes a repository in the current directory
2. git add . = stage all the changes made
3. git commit -m message = save the staged commit with a description
